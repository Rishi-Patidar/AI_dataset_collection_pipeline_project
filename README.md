
# AI Dataset Collection Pipeline Project

This project demonstrates how to plan, manage, and implement a simple AI dataset ingestion and validation pipeline.

Technologies:
Python
SQL (SQLite)
Tableau-style dashboarding (HTML charts)

Pipeline Stages:
1. Data Collection (synthetic image dataset)
2. Metadata creation
3. SQL ingestion layer
4. Dataset validation checks
5. Analytics dashboard

Run:
pip install -r requirements.txt
python main.py
